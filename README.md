Power BI Dashboard – User Story 5 & 6
Overview

Implemented interactive Power BI dashboards to analyze book data and generate insights.

User Story 5
Basic dashboard with book count, ratings, price, and availability
Single-page visualization layout
User Story 6
Price conversion (GBP → USD) and Price Tier classification
Subcategory creation and data cleaning
Advanced visuals: Tree Map, KPI Cards, Decomposition Tree, Scatter Plot
Interactive filters and navigation
Key Insights
Most books fall under Budget and Standard categories
Weak relationship between price and rating
Majority of books are in stock
Tools
Power BI Desktop
Power Query
Note

No traditional code is used; all logic is implemented within Power BI.
